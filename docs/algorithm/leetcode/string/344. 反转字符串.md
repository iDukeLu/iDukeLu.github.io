[344. 反转字符串](https://leetcode.cn/problems/reverse-string/description/)

编写一个函数，其作用是将输入的字符串反转过来。输入字符串以字符数组 s 的形式给出。

不要给另外的数组分配额外的空间，你必须原地修改输入数组、使用 O(1) 的额外空间解决这一问题。

## 求解

```go
func reverseString(s []rune) {
	if len(s) <= 1 {
		return
	}

	i, j := 0, len(s)-1

	for i < j {
		s[i], s[j] = s[j], s[i]
		i++
		j--
	}
}
```
