## MySQL 安装
```shell
docker run -d \
-p 3306:3306 \
-e MYSQL_ROOT_PASSWORD=123456 \
--restart=always \
--privileged=true \
--name=mysql \
mysql:8.0
```
更多参考：[MySQL Official Images | Docker Hub](https://hub.docker.com/_/mysql)

## ETCD 安装
```shell
docker run -d \
-p 2379:2379 \
-e ETCDCTL_API=2 \
-e ETCD_ROOT_PASSWORD=123456 \
--restart=always \
--name etcd \
bitnami/etcd:3.4
```
更多参考：[Bitnami package for ETCD | Docker Hub](https://hub.docker.com/r/bitnami/etcd)
